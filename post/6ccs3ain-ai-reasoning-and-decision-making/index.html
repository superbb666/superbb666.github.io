6CCS3AIN AI与策略决策

lecture 1

  1. sensors, actions, environment
  2. classify environment
  3. calculate the probability
  4. "clean the room"

lecture 2

  1. p 和 P 的区别
  2. 贝叶斯规则
  3. P的计算：α< >
  4. 可能性分布，乘积法则，贝叶斯规则，链式法则
    probability distribution可能性分布
    Product rule乘积法则
    Bayes rule贝叶斯规则
    Chain rule链式法则

lecture 3

  1. naive Bayes朴素贝叶斯
    朴素贝叶斯：所有变量都是独立的，是一种特殊的贝叶斯网络
  2. 贝叶斯网络
  3. Markov blanket马尔可夫毯
    parents + children + children’s parents：爹 + 孩 + 孩他爹
  4. Compact conditional distributions紧凑的条件分布——Noisy-OR
    Parents include all causes父系包含全部因素
    It assumes that inhibition of each parent is independent of
    inhibition of any other parents.否定状态下的条件独立
  5. 贝叶斯网络的推断方法：
    1. 枚举法
      利用概率的公式法则进行直接计算，不涉及随机数值
    2. 先验采样
      依照贝叶斯网络，由上至下由左至右，依次判断true/false，最后得到的是state状态集
    3. 拒绝采样
      ①. 与先验采样步骤相同
      ②. 遇到所要求计算的evidence（h，s）= false，则抛弃此次采样，重新开始计算
    4. 可能性权重
      ①. 与先验采样步骤相同
      ②. 当遇到evidence时，默认为true，并在此时计算权重，w = w x P
      ③. 通过每次的权重计算概率<1x0.72, 5x0.14>
    5. 吉布斯采样
      ①. 确定采样点（非evidence点，如M,T,C），此时进行随机采样并与0.5（both values equally likely）比较，获取初始state
      ②. 根据马尔可夫毯，计算每个sample的条件概率（遇到evidence，如H，S跳过）
      ③. 随机采样并与条件概率比较，判断true/false，并更新state
      重复②和③
      ④. 根据相同类型state的次数，计算所需的条件概率（此处与先验采样相同）

  1. MEU最大期待效用，maximin，maximax准则
  2. MDP马尔可夫决策过程，Bellman 贝尔曼方程
  3. POMDP 部分可见

  1. Dynamic Bayesian Network动态贝叶斯网络DBN
    sensor model: ↓ 传递模型
    transition model: → 过渡模型

    填条件法：
    填全集法：
  3. DBN具体内容
    Filtering:
    Prediction:
    Smoothing:
  4. 隐藏式马尔可夫模型HMM，卡尔曼滤波，粒子滤波

lecture 6

  1. game theory
  2. payoff matrix
  3. 四种模式
    1. dominant strategy：无论j选择了啥，i选择的R均为最大值
    2. Nash Equilbrium strategy：当j=R时，i的选择是最优的；当i=S时，j的选择也是最优的
    3. Pareto Optimal strategy：与所有状态相比，更改状态后，一方变好了，另一方一定变坏
    4. Maximise social welfware：求和最大

lecture 7

  1. conflict-free sets：冲突自由，没有任何attack
  2. admissible sets：可容许集，该{set}能够defend自身所有argument
  3. complete extensions：完整扩展，该{set}仅能够defend自身所有A，不能防御其他A
  4. Grounded extension：接地扩展，是minimal complete extension，T是所有子集的子集
  5. Preferred extension：优先扩展，是maximal complete extension，没有任何其他子集能包含T
  6. stable extension：稳定扩展，①. preferred extension，②. 内部元素能攻击子集外的所有所有人
  7. skeptical acceptance：怀疑接受，Ai在所有子集中均出现过才行(minimal)
  8. credulous acceptance：轻信接受，Ai在子集中，无论哪个子集，出现过即可
  9. 概念一：空集能够防御所有Argument
  10. 概念二：如果A没有被任何人袭击，那么他的防御者就是空集，也就是他就被所有人防御
  11. 概念三：何时complete extension存在空集：只要存在Ai没有被任何人攻击，那么complete extension 就不含空集

lecture 8

  1. argument game approach
  2. game tree，strategy for proponent&opponent(P&O)
  3. Argument game for grounded semantics:
    • Proponent starts with argument D.
    • Each move must reply to the previous move.
    • Proponent cannot repeat moves.
    • Proponent moves strict attackers, opponent moves attackers.
    • A player wins if and only if the other player cannot move.

lecture 9

  1. decision tree learning: Some, None, Full

  2. choosing an attribute(决策树学习算法)

  3. k-means算法

  4. 强化学习
